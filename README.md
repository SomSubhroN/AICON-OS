# AICON

A lightweight AI research workflow system for reproducible experiments.

## Features
- Experiment tracking
- Dataset management
- Environment control
- AI-aware scheduler
- Experiment Comparison

## Installation

git clone https://github.com/SomSubhroN/AICON-OS.git  
cd AICON-OS  
chmod +x install.sh  
./install.sh  

## Usage

aicon submit test.py --env testenv --dataset mydata  
aicon scheduler  
aicon jobs  

## Why AICON?

Reproducibility in AI research is broken.

AICON solves:
- messy experiments
- lost configurations
- inconsistent environments

## Example Workflow
1. Submit Experiment
2. Scheduler runs automatically
3. Logs stored
4. Compare Results
